Hexo-Server

hexo-server

Hexo 3 发布后, 服务器从主模块分离, 要使用服务器功能, 你首先得安装 hexo-server

一旦服务器被安装， 运行如下命令开启服务器. 你的网站默认运行在http://localhost:4000.当服务器运行, Hexo 会监视文件的修改并且自动更新, 所以没有必要重启服务器.

如果你想要改变端口或者遇到EADDRINUSEerros, 使用-p选项来设置一个不同的端口.

静态模式

在静态模式, 只有public文件夹里面的文件能被访问, 并且文件监视被禁用. 你在开启服务器之前必须运行hexo generate. 通常用于生产环境.

自定义IP

Hexo 服务器默认运行在 0.0.0.0. 你能覆盖默认IP设置.
